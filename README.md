## Installation

```bash
npx skills add ukyi/skills
```

## Included Skills

### base-ui-react

Production-tested skill for **Base UI** (`@base-ui/react` v1.2.0) — MUI's headless, accessible React component library.

Covers 36+ unstyled compound components including Dialog, Select, Popover, Tooltip, Menu, Tabs, Accordion, Toast, Combobox, and more.

**What's included:**

| File | Description |
|------|-------------|
| `SKILL.md` | Core skill — installation, render patterns, styling with data attributes, compound component API, form validation |
| `references/component-catalog.md` | Complete catalog of all 36+ components with props, usage patterns, and examples |
| `references/migration-from-radix.md` | Step-by-step migration guide from Radix UI to Base UI |
| `references/known-patterns.md` | Common patterns, gotchas, and production-tested solutions |

**When it activates:** Building UIs with `@base-ui/react`, implementing accessible components, migrating from Radix UI, using compound component patterns with render props, styling with data attributes and CSS variables.
